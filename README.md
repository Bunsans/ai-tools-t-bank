# AI Tools at T-Bank

Репозиторий для выполнения домашних заданий по курсу **AI Tools at T-Bank**.

## Описание курса

Курс предназначен для разработчиков, инженеров и технических специалистов, которые хотят осознанно использовать современные AI-технологии в своей профессиональной практике — от автоматизации рутинных процессов до создания интеллектуальных инструментов и сервисов.

Обучение строится на сочетании теоретических основ, практических демонстраций и самостоятельных заданий, что позволяет участникам не просто познакомиться с возможностями искусственного интеллекта, но и научиться применять их на реальных кейсах.

## Цели курса

- **Дать системное понимание** того, как развивались и функционируют современные AI-модели.

- **Научить использовать генеративные модели** для решения прикладных инженерных задач.

- **Освоить техники эффективного взаимодействия с LLM**: от промт-инжиниринга до создания собственных агентных решений.

- **Подготовить участников** к интеграции AI-инструментов в собственные проекты и рабочие процессы.

## Структура репозитория

В данном репозитории находятся выполненные домашние задания курса, каждое из которых размещено в отдельной директории.
